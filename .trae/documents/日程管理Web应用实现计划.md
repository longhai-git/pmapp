# 日程管理Web应用实现计划

## 项目概述
基于纯原生HTML/CSS/JS开发的日程管理应用，主打简洁轻便、快速记录、精准提醒。

## 技术栈
- **前端**: 纯原生HTML/CSS/JS，无外部依赖
- **富文本**: contentEditable + execCommand
- **数据存储**: localStorage
- **设计风格**: 新拟态轻质感，主色 #5b7cfa

## 文件结构
```
PMAPP/
├── index.html          # 主页面
├── styles/
│   └── main.css        # 全局样式（新拟态风格、响应式）
├── scripts/
│   ├── storage.js      # localStorage封装、导入导出
│   ├── app.js          # 应用入口、路由控制
│   ├── tasks.js        # 任务CRUD、六种视图
│   ├── inspiration.js  # 富文本编辑器、卡片墙
│   ├── timer.js        # 时钟、番茄计时、正计时
│   └── ui.js           # 弹窗、组件渲染
└── assets/
    └── favicon.ico     # 图标
```

## 实现步骤

### 阶段一：基础框架搭建
1. 创建 `index.html` 主页面结构（侧边栏、顶部栏、主内容区）
2. 实现 `main.css` 全局样式（新拟态风格、响应式布局）
3. 建立模块化JS文件结构

### 阶段二：数据存储模块
1. 封装 `storage.js`（localStorage读写、数据结构定义）
2. 实现数据导入导出功能（JSON格式）

### 阶段三：UI组件开发
1. 左侧导航栏（新增日程、首页、日程、项目、灵感、归档）
2. 顶部通栏（标题、实时时钟、日期）
3. 弹窗组件（任务表单、任务详情、编辑器）

### 阶段四：任务计划模块（核心）
1. 任务CRUD操作（添加、编辑、删除、完成）
2. 六种视图切换：
   - 月视图：日期网格，任务圆点标记
   - 周视图：周一至周日时间轴
   - 日视图：6:00-23:00纵向时间轴
   - 年视图：热力图展示
   - 日程视图：卡片列表
   - 项目视图：主任务卡片+子任务+进度条
3. 任务卡片操作（编辑、删除、延后、标记紧急/优先）
4. 超时提醒横幅
5. 归档功能（本周任务、已完成、未完成）

### 阶段五：灵感驿站模块
1. 富文本编辑器（加粗、斜体、下划线、删除线、图片、链接、音频）
2. 自动标题识别
3. 卡片墙展示（20字内标题）
4. 编辑/删除功能

### 阶段六：时钟与计时模块
1. 实时时钟（时:分:秒，大字体）
2. 番茄计时（25/15/45分钟模式，蜂鸣提醒）
3. 正计时（开始/暂停/重置）
4. 当前日期显示（星期+年月日）

### 阶段七：首页布局
1. 左侧：今日重点（自适应卡片网格，最多6张）
2. 右侧：周视图 + 本周进度环形图

### 阶段八：响应式适配
1. 移动端适配（≤600px汉堡菜单、单列布局）
2. 触控区域≥44px

## 数据结构设计
- **tasks**: 任务数组（id、标题、日期、时间、类型、状态、优先级、紧急度、项目、备注）
- **inspirations**: 灵感数组（id、标题、内容、创建时间）
- **projects**: 项目数组（id、名称、子任务列表）
